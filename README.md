# AP-final-Project
آرشام اسدی خوانساری
۴۰۴۱۳۴۰۲

وابستگی ها:
این پروژه به صورت کامل با استفاده از کتابخانه های استاندارد جاوا انجام شده و نیازی یه هیچ کتابخانه خارجی ندارد

نحوه کامپایل و اجرا:
برای اجرای صحیح بازی، دستورات را در ترمینال در 
پوشه اصلی پروژه اجرا کنید

دستور javac src/*.java برای کامپایل

دستور java -cp src main.GameMain برای اجرای بازی

لیست کنترل ها:
W / ↑  |  حرکت به بالا

S / ↓  | حرکت به پایین

A / ←  |‌  حرکت به چپ 

D / →  | حرکت به راست

Space  | شلیک گلوله

P      | توقف/ادامه بازی

Esc    | پایان بازی و بازگشت به منوی اصلی


توضیحات پایگاه داده:
برای ذخیره اطلاعات کاربران و تاریخچه بازی از فایل csv استفاده شده است.

این فایل به صورت خودکار در اولین اجرای برنامه ایجاد میشود. 

در فایل users.csv اطلاعات حساب های کاربری و تنظیمات آن ها ذخیره میشود .

ستون های آن شامل: نام کاربری، رمز عبور، بالاترین امتیاز کسب شده توسط کاربر، آخرین مرحله رسیده، تنظیمات صدای پس‌زمینه، صدای شلیک، صدای انفجار و صدای اتمام بازی است

تاریخچه بازی ها در فایل history.csv ذخیره میشوند.
ستون های آن شامل: نام کاربری، امتیاز نهایی، آخرین مرحله رسیده، تاریخ و زمان دقیق بازی و تنظیمات صدای فعلی است
